# Outlook Utilities (`src/server/utils/outlook`)

Specialized helpers for Microsoft's Graph API data structures.

## Key Files
-   **`message.ts`**: (Note: There is also an integration `message.ts`. This util file focuses on *helpers* for that integration).
-   **KQL Helpers**: Utilities for constructing safe Keyword Query Language strings.
-   **Response Normalization**: Converting Microsoft's `camelCase` response keys to our internal format.
